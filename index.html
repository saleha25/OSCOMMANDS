üöÄ Linux Tutorial: Learn the Basics

üìÅ Navigating Directories

cd: Change directory. Use cd .. to move up one level. For example:

  • cd /home/user/Documents: Go to the Documents directory.
  • cd ~: Go to your home directory.

üìç Print Current Directory

pwd: Displays the full path of your current working directory. This is useful when you're lost in the terminal!

üìÑ Listing Files

ls: Lists files. Some useful options:

  • ls -l: Long format (permissions, size, date).
  • ls -a: Show hidden files (files starting with a dot).
  • ls -h: Human-readable sizes.
  • ls -t: Sort by modification time.

Combine options, e.g., ls -laht.

üìù Creating & Editing Files

touch file.txt: Creates an empty file. For editing, you can use:

  • nano: A simple, beginner-friendly editor. nano file.txt
  • vim: A powerful, but more complex editor. vim file.txt
  • gedit: A graphical editor (if you have a GUI). gedit file.txt

üìÇ Creating Directories

mkdir folder: Makes a new directory (folder).

  • mkdir -p parent/child: Creates nested directories. If "parent" doesn't exist, it will be created.

üóëÔ∏è Deleting Files & Folders

Be careful with these! Deleted files are usually gone for good.

  • rm file: Deletes a file.
  • rm -r folder: Deletes a folder and its contents (recursively).
  • rm -rf folder: Force deletion (r = recursive, f = force). Use with extreme caution!

üìã Copying & Moving Files

  • cp source destination: Copies a file. cp file.txt newfile.txt
  • cp -r source_folder destination_folder: Copies a folder recursively.
  • mv source destination: Moves (or renames) a file or folder. mv oldname.txt newname.txt

üîç Finding Files

find path options expression: Powerful tool for finding files.

  • find . -name "file.txt": Finds "file.txt" in the current directory and its subdirectories.
  • find / -type d -name "folder": Finds directories named "folder" in the entire system.

üîé Viewing File Content

  • cat file.txt: Displays the entire file content.
  • less file.txt: Displays file content page by page; use arrow keys to navigate.
  • head -n 10 file.txt: Shows the first 10 lines.
  • tail -n 10 file.txt: Shows the last 10 lines.
  • tail -f file.txt: Follows the file; shows new lines as they are added (useful for logs).

üîê File Permissions

Linux has a robust permission system (rwx: read, write, execute) for user (u), group (g), and others (o).

  • chmod 755 file.txt: Changes permissions (7=rwx, 5=r-x).
  • chown user:group file.txt: Changes the owner and group of a file.

üë§ User Management

  • sudo adduser newuser: Adds a new user (requires sudo).
  • passwd user: Changes a user's password.
  • sudo deluser user: Deletes a user.
  • sudo usermod -aG group user: Adds a user to a group.

üì¶ Installing Packages

Package managers simplify software installation.

  • sudo apt install package_name: Installs a package on Debian/Ubuntu systems.
  • sudo dnf install package_name or sudo yum install package_name: Installs a package on Fedora/CentOS/RHEL systems.

üß™ Running Scripts

  • bash script.sh: Runs a shell script.
  • chmod +x script.sh; ./script.sh: Makes the script executable and runs it.

üñ•Ô∏è Monitoring System

  • top or htop: Shows running processes and resource usage.
  • df -h: Displays disk space usage in a human-readable format.
  • free -m: Shows memory usage in megabytes.

üì° Network Commands

  • ping google.com: Tests network connectivity.
  • ip a or ifconfig (deprecated): Shows network interface information.
  • netstat -tuln or ss -tuln: Lists open ports and network connections.

üìú Viewing Logs

Logs are crucial for troubleshooting.

  • tail -f /var/log/syslog: Follows the system log file (Debian/Ubuntu).
  • journalctl -xe: (systemd systems) View system logs.

üìå Process Management

  • ps aux: Lists all running processes.
  • kill PID: Terminates a process (find the PID using ps or top).
  • kill -9 PID: Forcefully terminates a process (use as a last resort).

üí° Terminal Shortcuts

  • Ctrl + C: Interrupts the current command.
  • Ctrl + D: Closes the terminal or ends input.
  • Ctrl + L: Clears the terminal screen.
  • Ctrl + Z: Suspends the current process.
  • Ctrl + A: Moves the cursor to the beginning of the line.
  • Ctrl + E: Moves the cursor to the end of the line.
  • Tab: Auto-completes commands and filenames.
  • Up/Down arrows: Scrolls through command history.
